# Build
`yarn build`
<br/><br/>

# Develop
1. Open two terminals
2. Run `yarn start-ts` in the first
3. Run `yarn start-bundler` in the second
<br/><br/>

# Test
`yarn build-test` or `yarn test`
<br/><br/>
